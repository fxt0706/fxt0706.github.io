    Welcome to ToddTail Blog

    由于大学本科结束，腾讯云的 1 元学生服务器到期，故将博客转移到了 Github Pages，也趁此将 Typecho 框架转为了 Hexo 框架。

    大学本科的四年里有了许多的变化，从最初的只对计算机、硬件技术以及其实践感兴趣，到现在能主动去接受较为生涩的哲学、经济学以及社会学方面的内容。从前在博客写文的时候，往往会重复着开坑-弃坑-填坑-弃坑的循环，这次这个新的开始，应当改变这个坏习惯了。

    几日后就要毕业了，这个新开的博客，作为对本科四年生活的祭奠，也作为一个下阶段新开始的见证了。
