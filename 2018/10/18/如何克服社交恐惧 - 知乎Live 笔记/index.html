    • 如何克服社交恐惧 - 知乎Live 笔记

    笔记内容来自知乎用户酷炫脑的 Live —— 如何克服社交恐惧。笔记仅作参考，详细内容请付费参与知乎Live。

    一、社交焦虑者的特征

    1. 高度自我关注，标准高
    2. 害怕犯错、丢脸、尴尬
    3. 负反馈可能变成惊恐发作
    4. 在公共场合压力大
    5. 有预期焦虑，提早很长时间焦虑
    6. 较扭曲的思维模式

    社交焦虑者的扭曲思考方式，与现实不符合，刻意考虑未来的社交活动中可以出错的地方和可能发生的突发事件。

    社交活动结束后，会认为自己有很多表现不好的地方，认为自己出错很尴尬，进行自我批判，且持续时间长。

    总结，社交焦虑者会倾向负面的思考方式，对自己的表现会产生不合理的负面想法，并由于过分的自我关注，认为他人必然会注意到自己的（假想的）负面表现，认为他人会对自己做出负面评价。

    绝大多数人在青春期（10-20岁）都会经历社交焦虑，主要都来自于自我关注，是常见的现象。

    二、社交焦虑的影响

    严重到一定程度，会影响到生活，工作，人际结交。

    可能伴随各种精神问题，惊恐发作，抑郁症，强迫症，以及生理症状，轻微头疼，晕眩等等。

    可能会产生药物、酒精依赖。

    三、社交焦虑的场合

    1. 当众发言
    2. 他人面前吃东西
    3. 他人面前工作
    4. 成为场合中的焦点
    5. 等等

    四、焦虑症占比

    焦虑症占美国20%人口，社交焦虑症占5%

    五、社交焦虑原因

    1. 生理方面，基因遗传
    2. 心理方面，幼年期被人欺负过，被忽视，做过丢脸的事，产生恐惧
    3. 环境方面，看到他人在特定场合中被嘲笑，习得性焦虑
    4. 父母从小过度保护，未习得社交技能，突然暴露在社交场合

    六、内向与社交焦虑的区别

    社交焦虑可能会希望参加社交场合，或者希望他人陪伴，但是由于种种原因进行了回避。

    区别：

    1. 内心是先天，社交焦虑是后天教育和环境经历
    2. 内向从独处或小范围交往中得到正能量，社交焦虑独处是如释重负，而非快乐，是回避行为
    3. 社交焦虑缺乏社交能力，预设自己会被人忽视，内向的人可以随时开启社交模式，社交的损耗可以通过和好友吃饭或独处补充能量
    4. 内向和社交焦虑的人都会在意别人的看法，尤其是重要的人，但社交焦虑会放大所有人的看法

    七、社交焦虑治疗

    临床最主流的疗法是认知行为疗法——把人的想法引向更理性的方向，不再回避带来焦虑的社交场合，产生不同于以前的反应。

    认知疗法主要用系统性脱敏疗法和现实生活中的暴露疗法。

    系统性脱敏疗法，让患者在安全放松的环境中，想象恐惧的社交场景，由治疗师处理恐惧情绪。由于想象时，周围环境时安全的，就可以让人的恐惧逐渐消散。

    现实暴露疗法，在治疗师的陪伴下，一起参加恐惧的社交场景。

    个人的疗法，主要以放松训练为主。例如腹式呼吸，瑜伽，冥想，通过自我反馈，降低焦虑程度。

    接纳和承诺疗法，让一个人学着接受焦虑和挣扎是生活的一部分，学会接受根据个人自我价值和意愿去体验生活，而不是被焦虑情绪主导去回避做生活的决策。这种主动的思维方式可以帮助人摆脱焦虑。研究发现，12周的接纳和承诺疗法可以明显改善生活质量。

    研究领域中，波恩大学的莫妮卡教授用催产素降低人的焦虑感。包括拥抱、接吻、做爱都会增加催产素。给特定焦虑障碍的人闻催产素，可以降低大脑负责焦虑情绪的杏仁核的活动。

    八、如何建立社会连接

    8.1 练习自我暴露

    焦虑者往往不太会分享自己的信息，所以往往很难开始一个新的连接，即很难卸下心房。练习分享，即主动向对方分享自己的看法，故事，一些生活的琐碎，以及评价对方的衣着，主动询问对方的情况，等等。建立亲密关系的前提，即是自我暴露。

    8.2 练习减少想象来自他人的批判

    焦虑者往往是在意他人对自己的评价，并过度估计交往对象的消极评价程度，认为他人只能看到自己的错误的一面。这种原因在于焦虑者认为自己自尊较低，思想上常常批判自己。

    8.3 接受自己

    这是非常重要的一点，接受自己，即认为自己是棒的，有价值的，能以富有同情心的方式看待自己的经历。这样可以使自己不再批判自己，意识到自身的价值建立在自身上，而不是建立在别人身上。

    接受自己不是一蹴而就的，需要长期去积累投入。一方面需要不断学习提高自己，一方面需要正确评估自己，着眼自己优秀的方面。每个人都有短板和各自的优点，把目光放在自己和他人不同的特有的优点，有利于接受自己。

    8.4 专注于当下

    这是克服焦虑的一个核心思维模式，因为焦虑大多来自于瞻前顾后，过度地去思考还未发生的事情并提早的配以消极思维去否定自己，进而会恶性循环，长期习惯于事情开始前就倾向放弃。专注当下，不去想过去和讲来。尝试以接受的心态接受当下，而不是一直审视和批判自己，并且学会识别自己的情绪和他人的情绪，以此为依据调整自己的行为（即培养情商）。

    情商高体现在五个方面：

    1. 监控和理解自己的情绪。不会忽视和赶走自己的情绪
    2. 情绪起来时，自我舒缓和处理情绪。
    3. 会利用情绪解决问题。
    4. 学会倾听和准确感受他人的情绪。
    5. 对他人的分享会表现出共情。

    九、社交的主要目的

    社交的主要目的是建立人和人之间的连接。焦虑会影响人的共情能力，即难以站在他人的角度进行思考。长期被焦虑情绪占用大脑，则很难有足够的思维和情感空间正确估计他人的想法和站在他人的立场。

    十、减少社交焦虑的八个自我调节方法

    1. 意识到焦虑是正常的。焦虑机制对于人的生存是至关重要的。
    2. 焦虑的想法不是现实。社交焦虑者的内心独白偏向消极。试着监控自己的想法并记录，在事情结束之后把消极想法与现实连接起来。
    3. 改变对生理反应的解释。对于紧张带来的生理反应，可以解释成是对事情的兴奋和期待造成的。
    4. 深呼吸。在焦虑开始时，尝试缓慢腹式呼吸，把注意力放在呼吸上，可以在短期内很好缓解焦虑（有实验证明）。平日可进行练习。
    5. 转移注意力焦点。把注意力从自己转移到事情或者社交对象身上，观注事情的进展以及他人的状态。
    6. 接纳社交中的不适感。暂时接纳不适感，进行社交任务后，会发现事情并没有想象中那么难，并因此得到信息，进入正反馈循环。
    7. 学会忍受不确定性。事情不是非黑即白的，不是所有人的都会认同我们，我们无法控制所有的事情。做自己内心的行为，体会事情中的乐趣即可。
    8. 转变立场。不做“参赛者”，而是以“裁判”的眼光来看待人和事，关注他人是否契合，表现是否恰当，或是事情是否正确运行。
